# Student Performance Analysis

## Project Overview

This project analyzes student academic performance using the Student Performance Dataset. The analysis was performed using Python in Jupyter Notebook with the help of data analysis and visualization libraries.

## Objectives

* Understand student performance patterns.
* Analyze the impact of study time on final grades.
* Examine the relationship between absences and academic performance.
* Identify key factors affecting student success.

## Dataset

Dataset Used: `student-mat.csv`

The dataset contains information about students, including:

* Gender
* Age
* Study Time
* Absences
* Grades (G1, G2, G3)
* Family and Social Factors

## Tools and Technologies

* Python
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib

## Analysis Performed

1. Data Loading and Exploration
2. Data Cleaning
3. Missing Value Analysis
4. Student Performance Analysis
5. Study Time vs Final Grade Analysis
6. Absences vs Final Grade Analysis
7. Correlation Analysis
8. Data Visualization

## Key Findings

* Students with higher study time generally achieved better grades.
* Increased absences were associated with lower academic performance.
* Previous grades (G1 and G2) strongly influenced final grades (G3).
